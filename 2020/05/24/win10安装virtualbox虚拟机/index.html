win10安装virtualbox虚拟机

服务器 虚拟机virtualbox virtualbox 虚拟机 2020/05/24

1. 下载

1.1 下载VirtualBox

下载地址：https://www.virtualbox.org/


（windows 版本下载）

1.2 下载centos

官网：https://www.centos.org/download/

镜像：http://mirrors.aliyun.com/centos/7/isos/x86_64/

1.3 CentOS中文网

https://www.centoschina.cn/

2.安装

2.1 开启虚拟化

https://www.tianqing123.cn/jy/519683.html

2.2 双击安装包，按步骤安装

3.创建虚拟机

3.1 点击菜单栏中的新建

3.2 选择类型/版本、内存大小、现在创建虚拟盘

3.3 选择虚拟机硬盘位置、硬盘大小、硬盘类型、固定大小

4. 安装CentOS7

4.1 选中刚安装的虚拟机，点击设置

4.2 选择【存储->没有盘片->选择CentOS系统盘->ok】

4.3 启动虚拟机

4.4选择安装时的语言

4.5 选择时区

4.6 选择磁盘分区

(默认设置即可，直接点击完成)

4.7 设置共享主机网络

(在配置中 勾选 常规-可用时自动链接这个网络)

4.8 选择[软件安装]相关选项

(初学者选择 GNOME桌面 和 开发选项)

4.9 点击开始安装

4.10 安装过程中设置 root密码、创建用户

(123456)

(admin,123456)

4.11 安装完成，点击重启按钮重启系统

4.12 接受许可条款，完成配置

4.13 登录，安装完成

5. 常见 初始化设置

5.1 进入CentOS7设置

5.2 关闭锁屏

(power->节电、空白屏幕->从不)

(privacy->锁屏->关闭)

5.3 设置 自动网络时间

5.4 自动登录

（先解锁，输入密码）

5.5 设置终端背景、文字颜色

6. 安装VirtualBox增强功能

6.1 安装

点击 设备->安装增强功能

(输入密码验证)

开始安装，等待…：

安装完成：

弹出光盘：

6.2 设置共享文件夹

选择 设备->共享文件夹

6.3 共享剪切板

(存在问题，暂时没生效)

Linux系列文章06 - 软件仓库和安装软件

服务器 Linux 软件包 软件安装 2020/09/02

软件包

概念

Centos下的“安装程序”被称之为 【软件包】，英文package

rpm：red hat package manager的缩写，红帽软件包管理器

软件包 与 windows下的exe文件的区别

  1. 软件包管理 其实 包括了 依赖关系的管理

    很少有软件可以在linux上单独运行，一个软件的运行通常要用到其他程序

    一个软件依赖其他软件，这个关系就是依赖关系

    依赖关系一般还有下层依赖关系，环环相扣

  2. 软件包不需要通过搜索引擎下载，所有软件包都被存放在软件仓库

    一般建议用户 使用离自己较近的软件仓库

切换软件仓库

​ 软件仓库地址存放在：/etc/yum.repos.d/CentOS-Base.repo

CentOS 官方源列表：

​ https://www.centos.org/download/mirrors/

修改CentOS默认yum源为国内yum镜像源

​ 详见：https://blog.csdn.net/inslow/article/details/54177191

包管理工具

终端的软件包管理命令一般是用【yum】

yum是CentOS默认的包管理工具

常用命令

更新软件包：

​ 例：yum update/upgrade

​ update 不删除旧包

​ upgrade 删除旧包

​ 在生成环境中最好使用update，防止删除旧包 而依赖出现问题

搜索软件包：

​ 例：yum search xxx

安装软件包：

​ 例：yum install xxx

删除软件包

​ 例：yum remove xxx

rpm命令 安装与卸载 软件包

​ 首先要去官网 下载 .rpm 文件

安装：

​ rpm -i xxx.rpm

​ yum localinstall xxx.rpm

卸载：

​ rmp -e 包名

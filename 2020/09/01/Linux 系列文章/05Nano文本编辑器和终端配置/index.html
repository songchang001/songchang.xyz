Linux系列文章05 - Nano文本编辑器和终端配置

Linux 服务器 Nano文本编辑器 服务器 终端配置 2020/09/01

Nano文本编辑器

nano是“毫微”的意思,也就是10的-9次方。

纳米科技的“纳米”就是 nanometer,由nano和 meter组成。

nano是”十亿分之一”, meter是“米

Nano的全称是 GUN Nano。

官网：https://www.nano-editor.org/

编辑器分类：

文本编辑器

文本处理器

启动nano编辑器

命令：nano + 回车

nano编辑器中的快捷键

nano的参数

打开文件名：nano file.txt

​ 如果文件 file.txt 不存在，就会创建一个空文件

激活鼠标：nano -m

激活自动缩进：nano -i

激活智能home键：nano -A

​ 通常按下home键 ，焦点会跳到一行最开始位置

​ 启用智能home键，会根据有无缩进跳转

例：nano -miA file.txt

nano的配置文件

​ nano配置文件叫 .nanorc，是隐藏文件，需要用ls -a 才能显示

​ Linux每个用户都可以在自己的家目录中创建.nanorc这个文件，nano启动前会读取此文件的配置

​ 如果家目录没有.nanorc文件，那么会用全局的配置文件

.nanorc文件 配置

​ 每一行一句配置语句，配置语句是以set 或 unset开头

​ set用于激活，unset用于关闭

​ set 和 unset后 接 要配置的项目

​ 例：

​ set mouse ： 开启鼠标，相当于每次附带 -m参数

​ set autoindent：激活自动缩进，相当于每次附带 -i参数

​ set smarthome：激活智能Home键

全局的nanorc 配置文件

​ 位置：/etc/nanorc （注意没有“.”）

​ 全局配置文件只能被root用户修改

​ /etc目录一般存放系统配置文件，一般为所有用户公用

​ /etc/nanorc 配置文件中 有所有 可以配置在 自己.nanorc 文件中的语句，但大部分都被注释掉了(以#开头为注释)

.bashrc 配置bash shell

.bashrc文件 一般都是默认存在的，存在于 ~/.bashrc

shell 是 外壳 的意思，是用于解释我们输入终端的各种命令的，是用户和操作系统之间的一个命令解释器

bash 是最常用的shell程序

.bashrc 就是 bash这个shell程序的配置文件

终端bash也有全局配置文件：/etc/bashrc

家目录下的.bashrc 文件 优先级 比全局 /etc/bashrc 文件高

.profile配置 非图形化终端

.bashrc 配置 图形化终端

.profile 配置 tty2~tty6 等非图形化终端界面

profile文件是会调用bashrc文件的，修改bashrc文件，也就是间接修改profile文件

修改.bashrc 和 .profile 文件，在用户下次登录系统时才能生效

立即生效方法：source .bashrc

rc文件

​ Linux中的配置文件，大多以”.”开头，多以“rc”结尾

​ rc 是 “runcomm”的缩写，即“run command”(运行命令)

​ rc是很多脚本类文件的后缀

​ 这些脚本通常在程序启动阶段被调用，.bashrc 是linux的bash shell启动后所运行的脚本

grep命令：筛选数据

Grep 是 Global Regular Expression Print 的缩写

grep 指令用于查找内容包含指定的范本样式的文件，如果发现某文件的内容符合所指定的范本样式，预设 grep 指令会把含有范本样式的那一列显示出来。

常规用法：

grep text file

text：要搜索的文本

file：要搜索的文件

例：

参数:

-i参数：忽略大小写

-n参数：显示文本所在行号

-v参数：显示文本不在的行

-r参数：在所有子目录和子文件中查找

高级用法：配合正则表达式

正则表达式字符及含义

-E参数：使用 正则表达式

linux中 egrep命令 效果等同于 grep -E

sort命令：为文件行进行排序

Linux sort命令用于将文本文件内容加以排序。

sort可针对文本文件的内容，以行为单位来排序

例：sort name.txt

-o参数：将排序结果写入新文件

例：sort -o new_name.txt name.txt

-r参数：倒叙排列

例：sort -r name.txt

-R参数：随机排序

例：sort -R name.txt

-n参数：对数字排序

wc命令：文件的统计

wc是word count的缩写，可用于统计文件行数、字符数、字节数

例：wc name.txt

-l参数：只统计行数

-w参数：只统计单词数

-c参数：只统计字节数

-m参数：只统计字符数

uniq命令：删除文件中的重复内容

cut命令：剪切文件的一部分内容

-c参数：根据字符数来剪切

​ 例：cut -c 2-4 name.txt ：剪切每行第2~4个字母

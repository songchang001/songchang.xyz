Linux系列文章08 - 查找文件

服务器 Linux 查找文件 2020/09/05

快速查找：locate命令

locate是英语定位的意思，locate命令用于定位要查找的文件

locate命令是搜索包含关键字的所有文件和目录

例：locate rename_file：查找名叫rename_file的文件

locate命令原理

locate命令不会对实际硬盘进行查找，而是在文件数据库中进行查找

强制更新文件数据库

locate命令的缺点：

  1. locate命令会列出所有在文件数据库中找到的内容
  2. locate命令不能找到一天内创建的文件(除非更新文件数据库)

深入查找：find命令

用法

find <何处> <何物> <做什么>

何处：在哪个目录中查找，会递归在子目录中查找

​ find命令可以指定查找目录，若未指定，会在当前目录及其子目录中查找

何物：找什么，其中<何物>为必填

​ 可以根据名称、大小、最近访问时间来查找

做什么：找到文件后，可以进行后续处理

​ 默认为打印文件路径

按名称查找

例：find -name “new_file”

​ find /var/log -name “syslog”

根据文件大小查找

例：find /var -size +10M ：查找/var中文件大小超过10M的文件

​ find /var -size -50k ：查找/var中文件大小小于50k的文件

​ find /var -size 50k ：查找/var中文件大小等于50k的文件

根据文件最近访问时间查找

​ -atime参数：access time的缩写，访问时间的意思

​ 例：find -name “*.txt” -atime -7 ：查找7天内访问过的txt文件

根据文件类型查找

仅查找目录类型：-type d

​ 仅查找文件类型：-type f

特点：完全匹配

操作查找结果

​ find -name *.jpg 等价于 find -name *.jpg -print

格式化打印

​ 例：find -name *.txt -printf “%p - %u\n”

​ %p：文件名

​ %u：用户名

​ \n：换行

删除查找到的文件

​ 例：find -name *.jpg -delete ：删除当前目录及子目录下所有jpg文件

​ 不会有确认提示，慎用

调用命令

例：find one -name “*.txt” -exec chmod 600 {} ;

​ 将one目录下的所有txt文件的权限修改为600

例：find one -name “*.txt” -ok chmod 600 {} \

​ 执行时 逐个确认
